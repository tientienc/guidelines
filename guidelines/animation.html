Guidelines & Assets

Animation

Animation serves as a great tool to help visualize and convey concepts that might be misunderstood when communicated in written content, or might take too long to explain.

Gifs

Gifs are a fun way to keep people’s attention while a screen is loading, to explain how things work in a quick and entertaining way, to communicate the subject of a document, or to simply make the recipient smile.

Naming

GIF_NAME.GIF

  • Use "_" to separate categories.
  • Use "-" to separate words WITHIN a single category.

Attributes

  • Gifs MUST loop forever.
  • Exceptions may apply.

Color system

  • Colors used must adhere to the LifeQ Color Palette.

Designing and exporting

1. Export design in Adobe Illustrator

  • Once design is finished, select "Full Layer".
  • Select burger icon to the right of Layers.
  • Select “Release to Layers (Sequence)”.
  • Save as .ai file.

2. Animate in Adobe After Effects

Setting up composition:
  • Size of gif will depend on the request.
  • Preset should be HDTV 1080 29.97.
  • The gif should be at full resolution (182 KB per 8 bpc frame).
  • Set the frames per second to 29.97 fps.
  • The background should remain white unless requested otherwise.
Importing:
  • Make sure that "Composition - Retain Layer Sizes" is selected when importing the Adobe Illustrator file.
Exporting:
  • Begin by adding gif to render queue.
  • Format should be Quicktime.
  • Video output:
    • Format Options: H.624
    • Channels: RGB
    • Depth: Millions of Colors
    • Color: Premultiplied (Matted)

3. Creating gifs in Adobe Photoshop

Import video from After Effects to Photoshop's timeline as an mp4.
To begin exporting, start with "Save for Web (Legacy)".

Preset:
  • Select "Gif"
  • Select "Selective"
  • Select "Diffusion"
  • Check "Transparency"
  • Select "No Transparency Dither"
  • Uncheck "Interlaced"
  • Colors: 256
  • Dither: 100%
  • Matte: #ffffff
  • Web Snap: 0%
  • Lossy: 0
Check "Convert to sRGB":
  • Preview: Monitor Color
  • Metadata: Copyright and Contact Info
  • Leave Color Table alone
Image Size:
  • Your custom size, at 100%
  • Quality: Bicubic
Animation:
  • Looping Options: Forever
